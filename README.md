# 🔐 RSA Encryption Project (Python + Streamlit)

##  Description
This project implements the **RSA encryption algorithm** using Python and provides a simple **Streamlit web interface** to encrypt and decrypt text messages.

---

##  Features
- RSA key generation (p, q, e, d)
- Message encryption
- Message decryption
- Interactive Streamlit UI

---

## Requirements
- Python 3.8+
- pip

---

##  Installation

### 1. Upgrade pip
```bash
python -m pip install --upgrade pip
```
### Install dependencies
```bash 
python -m pip install streamlit
```

### Clone the repository
```bash
git clone https://github.com/Idriss-Mghanem/RSA.git
```
### Run the project
1. Go to project folder
```bash 
cd RSA 
```
3. Run Streamlit app
```bash 
python -m streamlit run rsa.py
```
#### Open in browser

After running the command, open:

http://localhost or IP:8501
